# Value Overview

<p class="value-lead">TPF helps teams move from idea to production pipelines without getting buried in repetitive integration and deployment work.</p>

## At a Glance

<div class="value-glance">
  <div class="value-glance-item"><strong>Developer Joy</strong>Faster delivery with less glue code and safer refactors.</div>
  <div class="value-glance-item"><strong>Performance</strong>Reactive-first runtime behavior with lower operational overhead.</div>
  <div class="value-glance-item"><strong>Transport Choice</strong>One model, multiple client/server surfaces.</div>
  <div class="value-glance-item"><strong>Start Monolith, Split Later</strong>A practical architecture evolution path.</div>
  <div class="value-glance-item"><strong>Kube-Native</strong>Container-first, cloud-friendly operation model.</div>
  <div class="value-glance-item"><strong>Plugins, Not Glue</strong>Cross-cutting capabilities without domain coupling.</div>
</div>
